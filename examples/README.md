# Examples

Sample applications that consume Bijou64 from [GitHub Packages](https://github.com/Joel-hanson/bijou64/packages).

| Example | Description |
| ------- | ----------- |
| [kafka-counter-app](kafka-counter-app/) | Minimal Kafka producer/consumer using `bijou64-kafka-serializers` |

Each example includes a `settings.xml.example` for GitHub Packages authentication and a `pom.xml` you can copy into your own project.
